# Asynchronous Operations and Access Control

This project emphasizes asynchronous data handling and network operations using `Promise` and `async/await` syntax. I implemented robust data fetching logic utilizing both the native Fetch API and Axios. A critical part of this module was handling network exceptions and access control. Because the application transitioned to using Alta Software's secure internal APIs via VMware, I specifically designed logic to catch, manage, and gracefully resolve common authentication and authorization edge cases, such as `401 Unauthorized` and `403 Forbidden` HTTP responses.
