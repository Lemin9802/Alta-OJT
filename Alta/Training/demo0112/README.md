# Redux State Management and Internal API Integration

This module focuses on establishing a robust global application state using Redux and Redux Toolkit. The primary objective was to efficiently store, update, and retrieve data from the Redux store to ensure seamless cross-component usage. Initially built using mock APIs, the data layer was later transitioned to integrate with the company's real internal APIs. Accessing these live endpoints required a secure intranet connection via VMware. As a result, certain data-fetching features and Redux data flows in this module demonstrate handling real-world, secure enterprise data, though they may experience network timeouts if executed outside the company's internal network environment.
